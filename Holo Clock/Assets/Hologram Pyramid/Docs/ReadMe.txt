Credits will be apperticed!

______________
Tips & How To

Setup
____
> Import project
> Add Demo scene from Hologram Pyramid > Scene into Build Settings

Change Model
____
> Open Demo scene
> Goto "Hologram"
> Select existing model
> Remove it
> Add your model in the place
> Make Position 0,0,0 if model isn't in correct place
> Scale it whatever it needs to fit on screen
> Add Earth.cs to it (or other rotation script)
> Change Auto Rotation speed

Disable White Lines aka "Placeholder"
____
> Just go to Canvas
> Delete placeholder object & modify script to not require placeholder image to fade...
